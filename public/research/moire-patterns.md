# Moiré Patterns — 「規則的なものを2枚重ねたら何かが起きる」の数学

[2026-02-22のスケッチ](/diary/2026-02-22)で、シンプルなサイン波の円グリッドを作ってた。2/22の観察は「単純な規則の繰り返し」だったけど、これに**もう1枚同じ規則を回転させて重ねる**だけで、まったく違う何かが現れる。それがモアレ（Moiré）パターン。

## 何が起きるか

2つの規則的なパターン（ドットの格子、線の格子、サイン波）を**少しだけずらして重ねる**と、元のパターンには存在しない**大きな波模様**が出現する。

例：縞模様を2枚、わずかに角度をずらして重ねる → 干渉縞のような波が見える。元の縞のスケールよりずっと大きい波。

[Wikipedia: Moiré patternの図](https://en.wikipedia.org/wiki/Moir%C3%A9_pattern) を見てほしい。Moiréの典型例が並んでる。

これは光の干渉と数学的に同じ構造。2つのsin波を足し合わせると、低周波の「うなり」が出る現象（音響学のbeat phenomenon）と本質的に同じ。

```
sin(ax) + sin(bx) = 2 cos((a-b)x/2) sin((a+b)x/2)
```

`a-b` が小さいと、`cos((a-b)x/2)` が大きな波（モアレ）として見える。

## 2/22のスケッチに足すなら

今のスケッチは1枚のサイン波グリッド。これに2枚目を重ねる：

```js
function setup() {
  createCanvas(800, 800);
  noStroke();
}

function draw() {
  background(255);
  
  // 1枚目: 普通のグリッド
  for (let x = 0; x < width; x += 8) {
    for (let y = 0; y < height; y += 8) {
      let v1 = sin(dist(x, y, mouseX, mouseY) * 0.1);
      // 2枚目: 少しずらしたグリッド
      let v2 = sin(dist(x, y, width - mouseX, height - mouseY) * 0.1);
      
      // 重ね合わせ
      let v = (v1 + v2) * 0.5;
      fill(map(v, -1, 1, 0, 255));
      circle(x, y, 6);
    }
  }
}
```

マウスを動かすと、2つの「波の中心」が動いて、干渉縞がぐにゃぐにゃ動く。元の波の周期より大きなスケールの模様が、勝手に浮かび上がる。

## モアレを「効果」として使う

VJや映像表現で、モアレは2つの方向性で使える：

**1. 偶然性として** — 2つの規則的レイヤーを重ねるだけで、設計してない複雑さが出る。「因数分解しづらい」見た目を簡単に作れる。これは[2026-03-01](/diary/2026-03-01)で言ってた **「ズラシで因数分解しづらくする」** にド真ん中。

**2. 制御された干渉として** — モアレの周期は数式で計算できる：
```
モアレの周期 = (T1 * T2) / |T1 - T2|
```
T1, T2が元の2つのパターンの周期。これが分かると「ここに大きな波を出したい」を逆算できる。

## 印刷物との関係

新聞や雑誌の写真をスキャンすると、**勝手にモアレが出る**ことがある。あれは印刷の網点（halftone）と、スキャナーのセンサー格子のモアレ。意図せず出るからクリエイターには嫌われがちだけど、逆に言えば、**規則的なパターンを2つ重ねればモアレを「作品の一部」にできる**。

## カメラ・ディスプレイとの干渉

液晶ディスプレイにカメラを向けると、ピクセル格子とカメラセンサー格子のモアレが映る。これをスケッチで意図的にシミュレーションすると、「画面を撮影した映像」みたいな質感が出せる。VJで「スクリーン感」を出すテクニック。

## 自己モアレ（self-moiré）

1枚のパターンを「縮小して重ねる」とself-moiréが出る。例えば：

```js
// 同じスケッチを少し縮小して重ねる
push();
translate(width/2, height/2);
scale(0.97);
translate(-width/2, -height/2);
// もう一回同じ描画
pop();
```

これは1枚のレイヤーから自分自身でモアレを作る方法。フラクタルとモアレの中間みたいな、不思議な空間が出る。
